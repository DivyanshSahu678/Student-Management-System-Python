# 🎓 Student Management System (Python + MySQL)

A professional **Student Management System** developed using **Python** and **MySQL**. This project follows a modular structure and demonstrates CRUD operations, database connectivity, input validation, and object-oriented programming concepts.

---

## 🚀 Features

* ➕ Add Student
* 📋 View Students
* 🔍 Search Student
* ✏️ Update Student
* 🗑️ Delete Student
* ✅ Input Validation
* 🗄️ MySQL Database Integration
* 📦 Modular Project Structure
* 🧩 Object-Oriented Programming (OOP)

## ⚙️ Installation

1. Clone the repository.
2. Create a virtual environment.
3. Install the required packages.

```bash
pip install -r requirements.txt
```

4. Create a MySQL database.

```sql
CREATE DATABASE student_management_system;
```

5. Update the `.env` file with your MySQL credentials.

6. Run the project.

```bash
python main.py
```
